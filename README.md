## üß† Abstract
Sentiment Analysis, also known as Opinion Mining, identifies emotions or opinions expressed in textual data.  
This project implements **Deep Learning models (LSTM, BERT)** for text classification and compares them with traditional Machine Learning algorithms like **Decision Tree, Random Forest, and Logistic Regression**.

The LSTM model achieved **92.3% accuracy**, outperforming traditional ML models that ranged between 58‚Äì62% accuracy.

## üß© Methodology
1. **Data Collection:** Dataset of text reviews (from Kaggle).  
2. **Preprocessing:** Cleaning, tokenization, stopword removal, stemming.  
3. **Feature Extraction:** Using BERT embeddings and token sequences.  
4. **Model Training:** LSTM built with Keras; compared with ML models.  
5. **Model Evaluation:** Accuracy, Precision, Recall, and F1-score.  
6. **Deployment:** Flask-based web app for live sentiment prediction.
